research question

4_ Using a more specific query, are the new Chinese results less neutral?

Description Protocol Data

Description

Since it seemed from the previous protocol that the controversy didn’t exist in China, we though it was a problem of the query, so we changed it. This visualization is useful to grasp immediately the comparison between the two previous results (on the left we have the 50 results on google and on the right the 50 of Baidu of the previous visualization) and the new research with the new query (in the middle). What we could understand was that even though Social Credit System controversy was the most controversial query that we could search for, still the results are not at all similar to the western ones. They are just a little bit more debated than the previous query. To get to this point, we searched for multiple different queries on baidu.com, such as Social Credit System disadvantages, Social Credit System attitude, Social Credit System suggestions, Social Credit System opinions, before arriving to the one with the most controversial results that we could find, Social Credit System controversy. All the websites have some information about it but there are different approaches: neutral, so the website is just giving neutral information about the topic, generally explaining what it is; positive, so in this case the website is in favor of the system itself; negative, so the website is expressing itself against the system, being concerned and uncertain; then both positive and negative, so the article speaks for both parts. This first categorization helped us with the approaches categories for the next protocols. So, does the controversy exist in China? We still kept the duality in the next protocols to fully answer this question.

Protocol

Starting from the previous protocol, we implemented the research with a new query, again on Firefox browser and on incognito mode. We did the same process for these new 50 results and then we matched them with the previous results. Thanks to this more specific analysis it came to light that in Chinese websites the approach is anyway less controversial.
